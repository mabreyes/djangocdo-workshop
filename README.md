# Usage
```sh
$ git clone git@github.com:marcreyesph/djangocdo-workshop.git
$ cd djangocdo-workshop
$ pip3 install -r requirements.txt
$ python3 manage.py migrate
$ python3 manage.py collectstatic
$ python3 manage.py runserver
```
